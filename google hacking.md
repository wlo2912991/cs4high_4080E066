# google hacking指令碼
```
顯示google的總結信息
指令碼:info

顯示相關的網站
指令碼:related

搜索股票信息
指令碼:stocks

顯示詞語的意思
指令碼:define

搜尋包含關鍵字的文字
指令碼:intext

搜尋包含多組關鍵字的文字
指令碼:allintext

搜尋多組關鍵詞
指令碼:allintitle

搜尋 url 之中包含的文字
指令碼:inurl

搜尋 url 之中包含多組的文字
指令碼:allinurl

搜尋特定附檔名的文件
指令碼:filetype

搜尋特定的網站
指令碼:site 

搜尋所有和網頁有關的頁面
指令碼:link 

指令返回的結果是導入連結錨文本中含有搜索詞的頁面
指令碼:inanchor

搜尋數字
指令碼:numrange 

針對目錄列表的通用搜索
指令碼:intitle:index.of 

搜尋錯誤訊息
指令碼:error | warning

搜尋登入入口
指令碼:login|logon

搜尋用戶名
指令碼:username | userid|employee.id “your username is”

搜尋密碼
指令碼:password | passcode | “your password is”

搜尋管理員
指令碼:admin|administrator
```
